#CSS

---

```
selector {
	property: value;
}
```

---

```
body {
	background-color: red;
}
```

---

```
p {
	font-size: 1em;
}
```

---

```
.myclass {
	display: block;
	width: 50%;
	padding: 20px;
}
```

---

## Selectors can be:

- element
- class
- id
- complex selectors
